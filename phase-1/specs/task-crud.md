# Feature: Task CRUD

## Description
A console-based todo application that manages tasks in memory.

## Task Structure
- id: integer
- title: string
- description: string
- completed: boolean

## Features

### Add Task
- User can add a task with title and description
- Title is required

### View Tasks
- Show all tasks
- Display id, title, and completion status

### Update Task
- User can update title or description using task id

### Delete Task
- User can delete a task using task id

### Mark Complete
- User can mark a task as complete or incomplete
